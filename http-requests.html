HTTP Requests Format

In JavaScript, we make HTTP requests through what is known as AJAX (Asynchronous JavaScript and XML). I won’t go into the finer details of AJAX, but you can read up on it on the Mozila Developer Network. Essentially, we will be working with an XMLHttpRequest object through which we make calls to the API.

  1. In our application, we start by creating a request object with the line 
    var request = new XMLHttpRequest();
  2. In the next line, we specify the request type.
    request.open(requestMethod, queryURL + queryString);
    We pass req.open two parameters. The request method (GET,POST,PUT,DELETE) and the query. This is how we tell Twitch what information we want. We’ll need to cross-reference the official documentation to figure out how to fill this out. We'll go over how to do this in the next section.
  3. Remember our Client-ID? Twitch requires us to send a header containing the Client-ID with every API request. We do this with the following line:
    request.setRequestHeader("Client-ID", apiKey);
  4. Next, we tell our application what to do once we receive a response. We’ll come back to this later, but this is the format.
    request.addEventListener(“load”, function() {...} );
  5. Finally, we send our request with 
    request.send(null);

Now that we know how the general format of an API call, let's learn how to navigate the official Twitch API documentation and fill in request.open(...).
