Twitch API Documentation

Navigating the official documentation is simple if we know what we want. Recall that our goal was to get a list of STREAMS for the GAME HearthStone. We need to populate request.open(requestMethod, queryURL + queryString); with the correct method and query.

  1. We head over to the main page and find the section labeled Streams. Here we see that the valid request methods are GET. API calls are usually GET or POST requests.

    We can either pass the method directly as a string or through a variable like so var requestMethod = "GET";

  2. We now need to get our query. Near the top of the page, we find the base URL format of queries. Also note that data requests and responses are sent in JSON. This will be important later.

    All our queries will start with https://api.twitch.tv/kraken

  3. To get the final part of our query, we'll need to navigate to the specific API call.Technically, a query string of “/streams” is valid and would return a list of streams with the most viewers. However, we want streams specifically playing HearthStone.

    We use the parameter game shown to filter the results for hearthstone. We end up with the formatted string: /streams?game=Hearthstone%3A%20Heroes%20of%20Warcraft The quickest way to find the name of a game is to look at the URL of the game's page on Twitch.

  4. Our final function now looks like this. 

    request.open("GET","https://api.twitch.tv/kraken/streams?game=Hearthstone%3A%20Heroes%Warcraft")
